# clocktask
# 📌 Project Title

A simple web project that displays text centered on the page using HTML and CSS.

## 🚀 Features
- Center text horizontally and vertically
- Clean and simple design
- Easy to understand code

## 🛠️ Technologies Used
- HTML
- CSS

## 📂 Project Structure
project-folder/
│── index.html
│── style.css

## 📄 Code Example

### HTML
<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

### CSS
body {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
  margin: 0;
  text-align: center;
}

## ▶️ How to Run
1. Download or clone the project  
2. Open index.html in your browser  

## 📌 Output
Text will appear at the center of the page.
